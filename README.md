# CredAI 🚀

CredAI is a modern React-based web application built with Vite, designed to provide a fast, responsive, and user-friendly experience.

## 📌 Features

- Modern React UI
- Fast development with Vite
- Responsive design
- Reusable components
- Optimized production build
- Easy deployment to GitHub Pages

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- CSS
- Git & GitHub

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/credai.git
```

Navigate to the project directory:

```bash
cd credai
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will run at:

```text
http://localhost:5173
```

## 📦 Build for Production

```bash
npm run build
```

The production files will be generated inside the `dist` folder.

## 🌐 Deploy to GitHub Pages

```bash
npm run deploy
```
